减少上下文切换实战

¶第一步：用jstack命令dump线程信息，看看pid为53的进程里的线程都在做什么

sudo -u zouqiang usr/linux/app/jdk1.8.0_144/bin/jstack 53 > usr/linux/test_data/dump17

¶第二步：统计所有线程分别处于什么状态，发现300多个线程处于WAITING状态

grep java.lang.Thread.State ~/usr/linux/test_data/dump17 | awk '{print $2$3$4$5}' | sort | uniq -c

¶第三步：减少空闲线程数

避免死锁的几种常见方法

  • 避免一个线程同时获取多个锁
  • 避免一个线程在锁内同时占用多个资源，尽量保证每个锁只占用一个资源
  • 尝试使用定时锁，使用lock.tryLock(timeout)来替代使用内部锁机制
  • 对于数据库锁，加锁和解锁必须在一个数据库连接里，否则会出现解锁失败的情况
